## Description

Coding challenge

Manual check:

- Run the Application
- Visit: http://localhost:3000/
- Update the page multiple times and see (yes it is raw JSON):
  - There are only unique flights
  - Each request take no more 900ms

Adding new sources:

Update environment settings in `src/environments/environment.ts`

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
